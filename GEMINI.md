# Project: LLM Analysis Quiz

## Directory Overview

This directory contains the project statement for the "LLM Analysis Quiz" project. The goal of this project is to build a web service that can receive quiz tasks via a POST request, solve them, and submit the answers.

## Key Files

*   `Project Statement.txt`: This file contains the detailed requirements for the project, including:
    *   Instructions for setting up an API endpoint to receive quiz tasks.
    *   Details about the quiz format, which involves data sourcing, preparation, analysis, and visualization.
    *   Information about prompt engineering to resist or reveal a code word.
    *   Scoring and evaluation criteria.

## Usage

The contents of this directory are for the development of the "LLM Analysis Quiz" project. The `Project Statement.txt` file should be used as the primary source of requirements for building the application.
